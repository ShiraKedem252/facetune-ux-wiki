# Source: Reactions to Retake Feature & Headshots

**Researcher:** Shira | **Date:** February 2025 | **Sample:** n=11 qualitative interviews

Users perceive Retake (AI-generated headshots from existing photos) as a generative editing tool, not a re-photography solution. Adoption is blocked by user confusion about the feature's purpose and accuracy concerns. Headshots are valued as affordable alternatives to professional photography — but only if they look real.

## Three Levels of Authenticity Framework

Users evaluate Retake outputs across three dimensions:

| Dimension | User Question | Impact on Adoption |
|-----------|---|---|
| **Facial Features** | Does the nose/eyes/chin match my actual face? | Critical — inaccuracies break trust |
| **Expression** | Does it capture the mood/personality I want? | High — wrong expression feels fake |
| **Overall Look** | Does it look like me, just better? | Critical — must feel like authentic self |

## Key Inaccuracies Blocking Adoption

Users reported these failures in Retake outputs:

- **Facial feature distortion** — Asymmetrical face, changed nose shape, altered eye spacing
- **Expression mismatch** — Generated a smile when user wanted neutral, or vice versa
- **Proportions wrong** — Head position, neck angle, or shoulder width altered
- **Overall "look" off** — Output felt like a different person, not an enhanced version of self

## The Headshots Opportunity

Professional headshots cost $150-500. Retake offers an affordable alternative. But users only adopt if:
1. Output looks authentically like them
2. Feature is clearly labeled "AI Headshot Generator" (not "Retake")
3. Expectations are set upfront about accuracy limitations

## Adoption Barrier: Feature Confusion

Users couldn't explain what Retake does without extensive prompting. Is it a filter? A reconstruction? Traditional editing? This naming/positioning problem prevents trial.

## Methodology

- **Type:** Qualitative interviews + early insights doc
- **Participants:** 11 users shown Retake feature
- **Protocol:** Reaction, comprehension, willingness to use
- **Context:** Both professional and casual use cases
